# WebStreaming Nodejs


## Description

This project consists of a server and client application that interact with each other. The server fetches data from an API and streams it to the client, which renders the received data on a web page.

## Features

- Fetches data from an API
- Streams data from the server to the client
- Dynamically updates the web page with received data
- Provides start and stop buttons to control data streaming

## Prerequisites

- Node.js (version 19.9)
